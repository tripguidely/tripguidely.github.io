Setup: install a USA travel eSIM (step by step)

Do this on Wi-Fi before departure.

  1. Check compatibility: confirm your phone supports eSIM.
  2. Buy a plan: choose based on trip style (city vs road trip vs hotspot).
  3. Install the eSIM: follow provider instructions (QR/manual/app).
  4. Set the eSIM as your data line: keep your main SIM for calls/texts if needed.
  5. Disable data roaming on your main SIM: avoids accidental roaming charges.
  6. Activate when you land: turn on the eSIM line if required, then test data.

FAQ

Is an eSIM cheaper than roaming in the USA?

Often, yes. Roaming can be unpredictable. A prepaid USA eSIM is typically easier to control and set up quickly.

Can I keep my regular number while using a USA travel eSIM?

Usually yes. Keep your main SIM active for calls/texts if needed, and set the eSIM as your data line.

When should I install the USA eSIM?

Install it at home on Wi-Fi before your trip. Then activate when you land (if required) and test data right away.

Why does data not work after landing?

Most often, the phone is still using the primary SIM for data. Switch the data line to the eSIM in your cellular settings.
